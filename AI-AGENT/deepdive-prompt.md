# Deep Dive Prompt

Use `ChinaSiro/claude-code-sourcemap` as the only source of code truth.

Goals:

1. Explain what the target subsystem does.
2. Identify its main files under `restored-src/src/`.
3. Describe the execution flow in plain language.
4. Separate confirmed facts from inferred conclusions.
5. Prefer short evidence-backed explanations over speculation.
