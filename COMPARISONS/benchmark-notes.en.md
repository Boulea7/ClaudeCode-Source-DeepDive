# Benchmark Notes

Public benchmarks can add context, but they do not replace source analysis.

## What This Page Keeps

- model benchmarks can show model trends
- product benchmarks depend heavily on scaffolding, permissions, budgets, and tool integration
- public benchmark results are not enough to settle product design questions on their own

## Boundary

This repository uses benchmarks as background only. The main work remains the source-backed walkthroughs.
