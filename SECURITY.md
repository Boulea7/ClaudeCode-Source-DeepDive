# 安全说明

这个仓库主要是研究文档仓库，不是在线服务或应用代码仓库。

尽管如此，仍然可能出现需要私下处理的问题，例如：

- 意外提交了敏感信息
- 文档中包含不应公开的材料
- 外链指向了有风险的内容
- 某些安全相关表述可能误导读者

## 如何报告

- 请不要先开公开 issue 报告敏感问题
- 如果 GitHub 私密漏洞报告可用，请优先使用
- 如果该入口不可用，请通过仓库所有者的 GitHub 主页私下联系

报告时建议附上：

- 涉及的文件路径
- 对应 commit hash 或分支信息
- 问题的简短说明
- 你认为的影响范围

## 支持范围

- 当前只按 `main` 分支的最新状态处理安全相关问题
- 历史提交与旧版本文档默认不单独维护

## 不属于安全报告的内容

以下情况更适合走普通 issue 或 pull request：

- 一般性文档润色
- 非敏感的错别字或链接修正
- 对源码解读结论的普通讨论
